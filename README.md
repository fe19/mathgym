# MathGym

A simple, responsive web app for practicing math exercises.

## Features

- **Categories**: Addition, Multiplication, Division, or Mixed
- **Difficulty levels**: Easy, Medium, Hard
- **10 questions per run** with 4 multiple-choice answers
- **Timer** tracks how long each run takes
- **Leaderboard** with scores stored locally (localStorage)
- **Responsive design** — works on desktop and mobile

## Usage

Open `index.html` in any browser. No build step or dependencies required.

## How it works

1. Pick a category and difficulty
2. Answer 10 questions by clicking on one of the four options
3. See your score (0–10) and time at the end
4. Check the leaderboard to track your progress
